h1. TweetLinks

This project provides a script to tweet links from a file to a twitter account, reading the links and posting the title of the link and the link.

h2. Usage

<pre>
   Usage: tweetLinks.py [options] filename

   Options:
     --version             show program's version number and exit
       -h, --help            show this help message and exit
         -c FILE, --config-file=FILE
                                 Read this configuration file not default, oauth.cfg
</pre>
